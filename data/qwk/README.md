# QWK input datasets

Generated by `scripts/build_qwk_inputs.py` from
`outputs/test_eval/predictions/<task>/*.jsonl`.

One subdirectory per task model, each containing two files:

- `<task_model_slug>/<task_model_slug>.jsonl` — full schema, one row per (article, prompt).
- `<task_model_slug>/<task_model_slug>.csv`   — slim columns suitable for direct kappa input.

## Schema (per row)

| field | type | source |
|---|---|---|
| `article_id`        | string | persisted |
| `task_model_slug`   | string | persisted |
| `source_model_slug` | string | persisted |
| `prompt_idx`        | int    | persisted |
| `is_seed`, `is_best`| bool   | persisted |
| `prompt_sha256`     | string | persisted (jsonl only) |
| `parse_status`      | `ok` / `failed` | persisted |
| `gold_score`        | int 1-15 | persisted |
| `pred_score`        | int 1-15 or null | persisted |
| `gold_direction`    | int -1 / 0 / +1 | derived |
| `gold_confidence`   | int 1-5 | derived |
| `pred_direction`    | int -1 / 0 / +1 or null | derived |
| `pred_confidence`   | int 1-5 or null | derived |

## Decomposition rule

```
score 1..5    -> direction = -1, confidence = score
score 6..10   -> direction =  0, confidence = score - 5
score 11..15  -> direction = +1, confidence = score - 10
```

## How to use

```python
import pandas as pd
from sklearn.metrics import cohen_kappa_score

df = pd.read_csv('data/qwk/claude_sonnet46/claude_sonnet46.csv')
ok = df[df['parse_status'] == 'ok']

qwk_dir  = cohen_kappa_score(ok['gold_direction'],  ok['pred_direction'],  weights='quadratic')
qwk_conf = cohen_kappa_score(ok['gold_confidence'], ok['pred_confidence'], weights='quadratic')
```
